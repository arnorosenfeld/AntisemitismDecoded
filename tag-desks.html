Calibrate desk hotspots

Three entry kinds: hotspots are rotatable click-target rectangles (pre-filled from the current game state — edit only if you want to refine them), badge anchors are single points where the count circles should sit (click to place), and quads are 4-corner shapes whose corners drag independently (used for precise regions like the angled phone screen).

Hotspots — click-drag to draw, drag inside to move, corner handles to resize, green handle to rotate.
Badges — click on the image to place the point; click or drag to move.
Quads — click-drag to create an initial rect, then drag each corner independently. Delete key to reset.
