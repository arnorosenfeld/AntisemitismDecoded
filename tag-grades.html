Grade Sprite Tagger

Load the grade sprite PNG. Pick a grade from the buttons below, then click-drag on the sprite to draw a rectangle around that grade's cell. Each rectangle tags that grade's bounding box (as percentages of the sprite's natural dimensions), which is what CSS uses to crop the sprite to a single cell via background-position + background-size.

Tags persist in localStorage, so you can close and reopen. Use Copy JSON when done — paste into chat.
